# 小红书爆款拆解框架

> 拆解就是拆解。理解它为什么爆，就够了。

纯拆解框架。不仿写、不偷东西、不产出。

## 三步流程

```
第一步 → 拆标题：钩子类型 + 元素组成 + 吸引技巧 + 关键词安排 + 标签策略
第二步 → 拆正文：结构类型 + 语气口吻 + 开头结尾 + 表达技巧(9种+信号词) + 互动引导设计
第三步 → 拆情绪内核：它到底扎在读者哪个点上
         ↓
三步连起来看：标题/正文/情绪 方向一致吗？
```

每篇约 10 分钟。拆完5篇以上同领域的 → 翻出记录找共性，一段话总结规律。

## 快速开始
